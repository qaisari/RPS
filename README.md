# ✊✋✌ Rock Paper Scissors

A simple Rock Paper Scissors game built with JavaScript.

## Features

- Random computer opponent
- Win, lose, and draw detection
- Console-based gameplay
- Clean and simple game logic

## Technologies

- HTML
- JavaScript

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/qaisari/RPS.git
```

2. Open `RockPaperScissors.html` in your browser.

3. Open the browser's Developer Console (F12) to play and view the results.

## Game Rules

- ✊ Rock beats Scissors
- ✋ Paper beats Rock
- ✌️ Scissors beats Paper
